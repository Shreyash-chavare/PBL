# CodeSquad: Real-time Collaborative Code Editor

A real-time collaborative code editor where multiple users can edit code together and see changes instantly.

## ✨ Features
- Live collaborative editing
- Real-time synchronization
- Access to all LeetCode Problems
- Voice Chat function for room members

## 🚀 Getting Started

### Prerequisites
- Node.js
- npm
- MongoDB

### Installation
```bash
git clone https://github.com/Shreyash-chavare/PBL.git
cd PBL
npm install
```

### Running the App
1. Open two terminals
2. In the first terminal enter command
   
   ```bash
   cd ./Backend
   npx nodemon app.js
   ```
3. In the second terminal enter command
   
   ```bash
   cd ./Frontend
   npm run dev
   ```
4. Go to the link that appears in the second terminal or go to http://localhost:5173


## 🛠 Tech Stack
- **Frontend**: React, HTML, CSS, Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
- **Real-time Communication**: socket.io
